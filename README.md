# Face Recognition Attendance System

Automatically records attendance using face recognition.

## Requirements
Python 3.10, dlib-bin, face-recognition, face-recognition-models, Pillow, numpy, Click, flask, opencv-python

## Run
1. Open terminal in project folder
2. Activate virtual environment (optional)
3. Install dependencies (see Requirements)
4. Run:
   python app.py
5. Open browser: http://127.0.0.1:5000
